# Whatsapp Chatbot

## Getting Started
1. Install the dependencies:

    ```bash
    npm install
    ```

    or

    ```bash
    yarn install
    ```

2. Rename the `.env.example` file to `.env` and customize the environment variables as needed.

3. Start the server:

    ```bash
    npm run dev
    ```

Your Express server should now be up and running at http://localhost:5000.
